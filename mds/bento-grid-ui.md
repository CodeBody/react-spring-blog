# 便当盒 UI：bento-grid-ui 快速上手指南


## 📦 什么是 bento-grid-ui？


`bento-grid-ui`（便当盒布局）是一种源自 Apple 官网和现代仪表盘的高级布局模式。它通过 **非对称的网格结构 (Asymmetric Grids)**、**圆润的卡片嵌套** 以及 **错落有致的信息排布**，将复杂的各类信息碎片整合在一个极具美感的视觉框架内。它不仅是容器，更是一套**信息密度排布逻辑**。


## ⚙️ 如何安装？


你可以通过以下几种方式在你的项目中实现“便当盒”：


### 1. CSS Grid 官方原生方案 (推荐)
这是最灵活、最轻量级的生产级方案：
```bash
# 不需要额外库，直接利用 CSS Grid 的布局语义
display: grid;
grid-template-columns: repeat(12, 1fr);
grid-gap: 24px;
```


### 2. 通过 OpenSkills 动态加载
在 Agent 框架下一键同步布局范式：
```bash
npx openskills install frontend/bento --universal && npx openskills sync
```


### 3. Tailwind Bento 核心类名
在全局配置文件中添加便当盒核心比例：
```js
module.exports = {
  theme: {
    extend: {
      gridTemplateRows: {
        'bento': 'repeat(6, minmax(0, 1fr))',
      }
    }
  }
}
```

## 💬 如何通过对话使用？


调用 `bento-grid-ui` 的关键在于：**定义重要性分级与空间感**。


### 万能对话公式
> **[总容器尺寸]** + **[卡片优先级排布]** + **[网格跨度 (Span) 逻辑]**


---


### 🍎 苹果风产品落地页 (Apple-style Intro)

利用错落的网格展示产品的核心特性：

```markdown
请使用 bento-grid-ui 为我设计“产品亮点区”。风格要求：背景采用深灰色，网格按 2:1、1:1、2:2 的比例错开排布，所有卡片均使用巨大的圆角 (24px) 且内置柔和微弱的内阴影。
```


### 📈 高级金融仪表盘 (Finance Dashboard)

在极小空间内保持不同维度数据的可读性：

```markdown
请为我开发一个便当盒样式的“资金概览盘”。要求：左侧是一个跨越 2 行的大趋势图，右侧嵌套三个 1x1 的实时指标卡，并确保每个卡片之间的间距 (gap) 为 16px，整体呈现极简主义风格。
```


### 💼 AI 个人效率控制台 (Personal Hub)

整合各类琐碎的日程、天气、笔记：

```markdown
请将此任务列表重造成便当盒结构。风格要求：采用非对称网格，重点任务跨越 3 列展示详细描述，次要的天气和日程卡片采用 1x1 结构，背景使用低饱和度的莫兰迪色系。
```


### 🎨 多媒体创意展示 (Creative Gallery)

展示图片、视频与文案的完美交织：

```markdown
请使用 bento-grid-ui 设计“灵感墙”。风格要求：大的艺术作品跨越 2x2 格位，中等视频跨越 1x2，文案则见缝插针地填充在剩余空隙中，所有组件在悬停时都有稍微放大的呼吸感。
```


### 🛠 SaaS 复杂定价矩阵 (Pricing Grid)

通过视觉权重引导用户选择：

```markdown
请为我的 SaaS 产品设计便当盒定价模板。要求：主推方案 (Pro) 占据中间 2/3 的核心视觉位，免费版和企业版分别在左右翼次要位，所有卡片背景使用磨砂玻璃质感。
```
